# 補習小天地 StudyPanda

小學英文數學練習平台 - Primary School Math & English Practice

## 功能 Features

- 📚 數學 Math: P1-P6 年級題目
- 📝 英文 English: P1-P6 年級題目  
- 🌐 三種語言: 繁體中文、簡體中文、English
- 📊 學習進度追蹤 (localStorage)
- 🎯 10題/次，即時計分

## 使用方法 How to Use

1. 開啟 `index.html` (用瀏覽器直接打開)
2. 選擇語言 (Language)
3. 選擇科目 (數學/英文)
4. 選擇年級 (P1-P6)
5. 按「開始練習」
6. 完成10題後提交
7. 查看成績

## 技術 Tech

- Plain HTML/CSS/JS (no framework)
- localStorage for progress
- AI-generated questions logic
